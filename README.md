<h1>Star Map Navigation Simulation</h1>

<h2>Description</h2>
Procedural starmap generator which creates paths between stars that can be navigated through by pathfinding algorithms.<br><br>

The "Generation" menu allows the user to create wide ranges of galaxies with different shapes, sizes, distance multipliers and more!<br><br>
<img width="800" height="450" alt="StarmapGeneratorClip" src="https://github.com/user-attachments/assets/c6d703ee-24b8-4290-a47a-e8f84dde18f3" /><br>

Generated stars can be selected with the mouse to show their attributes (name, type, mass) in the "Pathfinding" menu,<br> and selected stars can be set as start and end points to pathfind between them.<br><br>
